# Ecosystem Dynamics Lab

Welcome to the **Ecosystem Dynamics Lab** GitHub organisation.

## 🌿 Who we are

The Ecosystem Dynamics Lab is a research and teaching group focused on understanding how ecological systems change across space and time. Our work integrates:

- Spatial ecology
- Biodiversity and conservation science
- Environmental monitoring and modelling
- Applied data science using R


## 💻 What this GitHub organisation is for

This organisation is used to support the lab’s work by hosting:

### 📊 Research projects
- Reproducible analyses
- Data processing workflows
- Manuscript-associated code
- Collaborative project repositories

### 🎓 Teaching materials
- R tutorials and course materials
- Practical exercises and assignments
- Example datasets and analysis pipelines

### 🔁 Reproducible science
We aim to follow best practices in open and reproducible research, including:
- Version-controlled code
- Transparent workflows
- Reusable analysis pipelines
- Clear documentation

## 🧰 Tools and languages

Most of our work is conducted using:

- **R / RStudio**
- Git and GitHub


## 📁 Repository structure

Repositories in this organisation typically follow one of these patterns:

- `project-name`: Research projects or analyses
- `course-code_topic`: Teaching materials
- `utils` or `lab-tools`: Shared functions and reusable code

Each repository should include its own README describing the specific project.

## 🤝 Contributing

Members of the lab are encouraged to:

- Use clear and descriptive commit messages  
- Document code and workflows  
- Create reproducible analysis pipelines  
- Use issues and pull requests for collaboration  

If you're new to GitHub or Git workflows, guidance and templates will be provided.

## 🔐 Access and collaboration

Most repositories are:
- Private during active development
- Made public where appropriate (e.g. publications, teaching resources)

External collaborators may be granted access on a per-project basis.
